# Android App Template

Private template repo for Android apps.

## Quick start

1. Click **"Use this template"** → Create new private repo
2. Find/replace `com.myapp.app` → your package name
3. Update `app_name` in `res/values/strings.xml`
4. Update `APP_NAME` in `.github/workflows/build.yml`
5. Push to `main` → APK appears in GitHub Releases automatically

## What's included

- ✅ Fixed release signing (no "App not installed" on updates)
- ✅ VI/EN bilingual toggle in MainActivity
- ✅ GitHub Actions: auto-build + upload APK on every push
- ✅ Dark theme, programmatic UI (no XML layouts)
- ✅ `AGENTS.md` — AI developer rules

## Rules

See `AGENTS.md` for mandatory rules all developers (human and AI) must follow.
